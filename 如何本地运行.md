# 如何本地运行（解决 CORS / 看不见桌子）

页面必须通过 **HTTP 服务器** 打开，不能直接双击 `index.html`，否则 GLB 模型会因 CORS 被拦截，看不到桌子。

---

## 方式一：用 VS Code 的 Live Server（推荐，无需装 Python/Node）

1. 在 VS Code 左侧点击 **扩展**（或按 `Ctrl+Shift+X`）
2. 搜索 **Live Server**
3. 安装 **Live Server**（作者：Ritwick Dey）
4. 在左侧文件列表里 **右键** `index.html`
5. 选择 **Open with Live Server**
6. 浏览器会自动打开（一般是 `http://127.0.0.1:5500`），此时就能看到桌子

---

## 方式二：用 Node.js 启动（若已安装 Node）

在终端（PowerShell 或 CMD）里，进入项目目录后执行：

```bash
npx -y http-server -p 8000 -c-1
```

然后在浏览器打开：**http://localhost:8000**

---

## 方式三：用脚本自动选 Node 或 Python

双击运行项目里的 **`start-server.bat`**：

- 有 Node.js 会用 Node 起 8000 端口
- 没有 Node 会尝试用 Python
- 都没有会提示你装 Live Server 或 Node

---

## 总结

| 方式           | 需要安装     | 操作 |
|----------------|--------------|------|
| Live Server    | 无（仅 VS Code 扩展） | 右键 index.html → Open with Live Server |
| start-server.bat | Node 或 Python | 双击 bat 文件 |
| 命令行         | Node.js      | `npx -y http-server -p 8000 -c-1` |

**不要** 用 `file:///F:/...` 或直接双击 HTML 打开，否则模型加载会失败。
